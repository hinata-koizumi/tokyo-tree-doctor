# ユーティリティ

Tokyo Tree Doctorのユーティリティツール

## ディレクトリ構造

- `data_collection/` - データ収集ツール
- `image_processing/` - 画像処理ユーティリティ
- `reporting/` - レポート生成ツール

## 機能詳細

### データ収集
- ドローン画像自動収集
- センサーデータ取得
- 外部API連携
- データ品質チェック

### 画像処理
- 画像前処理
- 品質向上
- 形式変換
- メタデータ抽出

### レポート生成
- 分析結果レポート
- 統計レポート
- 可視化チャート
- PDF/Excel出力

## 使用方法

各ユーティリティには使用方法とサンプルコードが含まれています。
