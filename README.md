# Tokyo Tree Doctor

AIとドローン技術を活用した包括的な樹木健康監視・診断システム。

## プロジェクト構成（現状）

```
tokyo-tree-doctor/
├── ml/                  # 機械学習コンポーネント
│   ├── assets/          # 機械学習アセット
│   │   ├── images/
│   │   └── videos/
│   ├── config/          # 機械学習設定
│   ├── data/            # 機械学習データ
│   ├── docs/            # 機械学習ドキュメント
│   └── tests/           # 機械学習テスト
│
└── software/            # ソフトウェアコンポーネント
    ├── assets/          # ソフトウェアアセット
    │   ├── images/
    │   └── videos/
    ├── config/          # ソフトウェア設定
    ├── data/            # ソフトウェアデータ
    ├── docs/            # ソフトウェアドキュメント
    └── tests/           # ソフトウェアテスト
```
