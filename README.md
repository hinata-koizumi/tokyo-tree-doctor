# Tokyo Tree Doctor

AIとドローン技術を活用した包括的な樹木健康監視・診断システム。

## プロジェクト構造

```
tokyo-tree-doctor/
├── assets/          # 画像、モデル、動画
├── config/          # 設定ファイル
├── data/            # データストレージ（生データ、処理済み、モデル）
├── docs/            # ドキュメント
├── notebooks/       # 分析用Jupyterノートブック
├── scripts/         # ユーティリティスクリプト
├── src/             # ソースコード
│   ├── ai_analysis/     # AI分析モジュール
│   ├── dashboard/       # Webダッシュボード
│   ├── data_pipeline/   # データ処理パイプライン
│   └── drone_diagnosis/ # ドローン診断
└── tests/           # テストファイル
```

## 機能

- AI駆動の樹木健康分析
- ドローンによる画像収集
- リアルタイム監視ダッシュボード
- リスク評価とスコアリング
- 樹木画像処理用データパイプライン
